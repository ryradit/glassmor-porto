### Colors
Modify the color scheme in `globals.css`:
```css
:root {
  --primary: #9333ea;
  --secondary: #a855f7;
  --accent: #c084fc;
```

Update personal information in each component:
- Personal details in `AboutSection.tsx`
- Skills in `SkillsSection.tsx`
- Projects in `ProjectsSection.tsx`
